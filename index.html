A FEW COMMON DATA TYPES

INTEGER DATA TYPE

The integer data type can hold numbers, specifically integers of any length or value. They are not able to store decimals however.

x=1
y=234
z=567890000000

All of the above are examples of integer data types.



Uses of Integer Data Type


  • Used in cases of whole number calculations
  • Great for counter operations
  • Easy to handle and use numeric operators on

FLOAT DATA TYPE

The float data type can hold numbers, both integers and decimal values of any length. Even the integers are pointed in a decimal format though which marks that the variable is of float type.

x=1.0
y=1.23
z=456.7890123

All of the above are examples of the float data types.



Uses of Float Data Type


  • Used in cases of decimal calculations
  • Great for fundamental attribute factors
  • Handles complex conditions better

STRING DATA TYPE

String data types are able to work with characters. They do not work with numeric operators and are helpful in handling words, alphabets, and special characters.

name="sid"
roll="18dse002"
sec="B"

All of the above are examples of string data types.



Uses of String Data Types


  • String Data types help a lot with manipulating words
  • Only way for working with anything that is not numeric and has to be worked in chains
  • Perform special string operations

BOOLEAN DATA TYPE

Boolean data types are able to work with the truth of different statuses of variables. They are the ultimate check of any value and are great for control flow statements.

truth=True
truth=False
truth=(5>3)

All of the above are examples of boolean data types



Uses of Boolean Data Types


  • Boolean data types are great for comparing values
  • Make the core of control flow statements
  • Boolean data also help in changing the status of an action

LIST DATA TYPE

List data types are able to work with an array of items that are indexed. Each item can go through operations individually or the whole set can be operated on altogether.

numbers=[0,1,2,123]
names=["sid","zahir","chris"]
values=["sid",123]

All of the above are examples of list type data



Uses of List Data Types


  • Lists are helpful in operating on a number of items together
  • They can reference to one item in a group through index
  • Selection based on condition can be done as well
